# Tic Tac Toe

A Tic Tac Toe built with Html Css and Js

## Features

- **Reset Button** - It resets the game witha click of a button
- **Draws a line** - It draws a line when a player wins



## How to Play

1. Click on one of the grids
2. Same goes for the second player
3. Get either cross or circle to get three in a row eith in a row collom or diagonal
